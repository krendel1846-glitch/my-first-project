- генерация разнесена на вкладки: Параметры / Выбор вкусов / Результаты

Как обновить на GitHub:
1. Загрузить index.html, manifest.webmanifest, sw.js и всю папку icons с заменой.
2. Подождать 1-2 минуты.
3. На iPhone открыть сайт в Safari.
4. Если иконка была добавлена раньше: удалить старую иконку с экрана Домой, затем добавить снова.
5. Если на iPhone все равно показывается старый вид: открыть Safari -> закрыть вкладку сайта -> открыть снова по ссылке.
