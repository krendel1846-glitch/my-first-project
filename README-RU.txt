# Hookah Mixology Engine Pro — веб‑app для iPhone

## Что это
Это версия твоей программы, которую можно:
1. выложить по ссылке,
2. открыть на iPhone в Safari,
3. добавить на экран «Домой» как приложение.

После первого открытия сайт будет работать офлайн для основных функций.

## Как выложить без программирования
### Вариант A — Netlify Drop
1. Открой на Windows: https://app.netlify.com/drop
2. Перетащи СОДЕРЖИМОЕ этой папки в окно браузера.
3. Получишь ссылку вида `https://...netlify.app`
4. Открой ссылку на iPhone в Safari.
5. Нажми «Поделиться» → «На экран Домой» → включи «Открывать как веб‑приложение» → «Добавить».

### Вариант B — GitHub Pages
1. Создай репозиторий на GitHub.
2. Загрузи в него все файлы из этой папки.
3. Включи Settings → Pages.
4. Открой выданную ссылку на iPhone.

## Важно
- Основные функции работают локально и офлайн после первого открытия.
- OCR по фото и импорт Excel могут требовать интернет, потому что внешние библиотеки подгружаются отдельно.
- Для лучшего офлайна потом можно добавить локальные vendor‑файлы.
