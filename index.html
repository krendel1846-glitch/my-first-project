Hookah Mixology Engine Pro
Single-file rule-based mini app • без API • локальная логика миксологии

Параметры генерации

Если режим ручной — программа строит несколько разных концепций на одном и том же наборе 2–5 вкусов: меняет тело, роли и процентовки, а не просто перебирает одни и те же схемы. Если режим авто — подбирает логически сильные сочетания из базы.

Движок полностью локальный: без API, без сервера. Совместимость считается по правилам ролей, телу микса, сенсорике, рискам подавления и читаемости.

Импорт

Поддерживается табличная вставка из Excel/Google Sheets и CSV/TSV. Колонки: Бренд, Вкус, Описание, Крепость, Тип вкуса, Вес.

Импорт файла

Поддерживаются .xlsx, .xls, .csv, .tsv, .txt. Excel читается через SheetJS из CDN, CSV/TSV — локально.

Поиск вкуса по фотографии

Сфотографируй банку или пачку табака, вытащи текст с упаковки и найди совпадение в базе. Хорошо работает для названий вроде Blackburn Malibu.
