- обновлены баннер и иконки приложения.

Как обновить:
1. Загрузить файлы в GitHub с заменой.
2. Нажать Commit changes.
3. Подождать 1–2 минуты.
4. Обновить сайт 2–3 раза.
5. Если старая версия держится на iPhone — удалить иконку с Домой и добавить заново.
