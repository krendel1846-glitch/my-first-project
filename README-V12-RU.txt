v12 — English-first Brand Lock OCR

Что изменено:
- OCR теперь сначала работает в режиме ENG, затем при необходимости делает fallback ENG+RUS.
- Добавлен жёсткий brand lock: если бренд уверенно найден, поиск ограничивается вкусами только этого бренда.
- Усилена нормализация OCR-ошибок для английских брендов и названий.
- Добавлены алиасы для Overdose, BlackBurn/Blackburn, Must Have, Darkside и других брендов.
- В результатах поиска выводится определённый бренд и сообщение, что поиск ограничен этим брендом.
